<H1>FinanceiroBackend

<h4>FinanceiroBackend é um sistema de gestão financeira pessoal desenvolvido em Java com Spring Boot. Ele disponibiliza uma API RESTful para gerenciar contas, lançamentos financeiros, usuários, centros de custo e bancos, com operações completas de criação, atualização, listagem, exclusão e pagamento de lançamentos. O sistema utiliza banco de dados em memória H2.
